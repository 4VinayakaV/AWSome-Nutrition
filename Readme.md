Team Name: AWSome Nutrition

Title: AI-Powered Nutrition Tracking Application on AWS.

**Overview**:

AWSome Nutrition is a cloud-native, AI-powered diet management application built using a fully serverless architecture on AWS. The platform enables users to generate personalized diet plans, track meals, analyze nutritional information from images, and interact with an AI chatbot for dietary guidance.
The system integrates modern cloud technologies and artificial intelligence to deliver a scalable, responsive, and personalized user experience.

**Note:** This repository is a cleaned and public-facing version of a collaborative academic project originally developed over multiple weeks.
